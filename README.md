# 🎣 후킹 되거나, 훅 가는 시대 — 강의 덱

스폰지클럽 AI 공유회 강의 덱(43장). **`index.html` 하나로 어디서든 열리는** 자체완결 파일(영상·이미지 전부 내장).

## 진짜 링크로 여는 법 (GitHub Pages · 무료)
1. github.com 에서 새 저장소 `hooking-deck` 생성(Public)
2. 이 폴더를 push:
   ```bash
   cd ~/Desktop/hooking-deck
   git remote add origin https://github.com/Koo-bon/hooking-deck.git
   git branch -M main && git push -u origin main
   ```
3. 저장소 → **Settings → Pages → Branch: main / (root)** 선택 → Save
4. 잠시 뒤 이 주소가 열림 (클라우드 로그인 없이 누구나):
   **https://koo-bon.github.io/hooking-deck/**

## 로컬로 바로 보기
`index.html`을 브라우저에 드래그하면 됩니다.
